# SYSUER 美化辅助增强脚本

中山大学（SYSU）官网、教务系统、在线教学平台综合辅助油猴脚本，旨在通过自动化操作和页面净化，大幅提升中大学子的网页端使用体验。

---

## ✨ 核心功能

### ⚙️ 可视化配置面板
* **便捷入口**：页面右下角常驻悬浮按钮（⚙️），也可通过油猴扩展菜单呼出，随时随地修改配置。
* **中大专属主题**：UI 深度定制，面板与提示框均采用中山大学标志性**“中大绿”（`#005826`）**配色，沉浸感十足。
* **即时生效**：所有配置项（如账号密码、功能开关）均可通过图形化面板修改，持久化保存在本地。

### 🚀 自动化登录与网络优化
* **自动登录 NetID**：在 CAS 统一身份认证页面自动填写账号密码并提交。
* **跳过 MFA 验证**：自动注入信任 Cookie，跳过繁琐的多因素认证页面。
* **智能 WebVPN 跳转**：在外网访问受限的校内资源（如 `appgw` 或 `visitor` 拦截页）时，自动替换域名跳转至 WebVPN。
* **无缝中转**：在教务系统、LMS、Portal 等系统的中转页，自动点击“登录”按钮，减少鼠标点击次数。

### 🎓 页面净化与美化
* **去除冗余元素**：精简教务系统（jwxt）及官网中无用的头部、底部、面包屑导航和调查问卷。
* **排版优化**：自动拓宽成绩查询、个人培养方案等页面的表格展示区域，提升信息获取效率。

### ⏩ 教学平台 (LMS) 视频速通
* **后台秒过**：检测到在线教学平台（`lms.sysu.edu.cn`）的视频任务时，自动高频提交视频播放进度，数秒内完成。
* **自动连播**：视频进度提交完成后，自动点击“下一页”进入下一个学习任务。

## 📦 安装与使用指南

### 1. 安装脚本管理器
请确保你的浏览器（Chrome / Edge / Firefox 等）已安装 **[Tampermonkey (油猴)](https://www.tampermonkey.net/)** 扩展。

### 2. 安装本脚本
在 Tampermonkey 中新建一个用户脚本，将本脚本的完整代码粘贴进去并保存（`Ctrl + S`）。

### 3. 初始配置
1. 访问任意受支持的中大网站（如教务系统或门户）。
2. 点击页面右下角的 **⚙️ 绿色悬浮按钮**，或点击油猴扩展图标选择 **“⚙️ 脚本设置”**。
3. 在弹出的面板中：
   * 输入你的 **NetID 用户名** 和 **密码**（用于自动登录）。
   * 根据个人喜好，开启或关闭“自动跳转 WebVPN”、“页面美化”、“视频自动速通”等功能。
4. 点击 **“保存设置”**，刷新页面即可享受增强体验。

* **数据纯本地存储**：你的 NetID 账号与密码仅通过油猴自带的 `GM_setValue` API 加密存储在你本人的浏览器本地，**绝对不会**上传至任何第三方服务器。
